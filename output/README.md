# output/

This directory stores the preprocessed dataset and model evaluation results.

## Structure
- `X_*.npy`, `y.npy` → Preprocessed skeleton features and labels generated by the `preprocessing/` pipeline.
- `label_map.json` → Class name to integer index mapping.
- `charts/` → Visualizations and accuracy plots generated by `main.py`.
- `model_results.json` → Cached evaluation metrics from multi-model comparison runs.

## Notes
- This directory is populated automatically during preprocessing and training.
- Large `.npy` files are generally excluded from Git.
